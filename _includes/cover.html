{% comment %} Figure include adds an image marked up using bootstrap, https://getbootstrap.com/docs/5.1/content/figures/ Options: - "img" = the filename of the image located in the /images/ folder of your repository, or full url to external image - "alt" = description of the image contents for the alt text - "caption" = text for caption of the figure (optional) - "width" = manually set width using options such as "50%" (optional) {% endcomment %}
